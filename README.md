Takes an .mtl file and removes the reference to child folders for "map_Kd" and "map_d".
Useful when you want to store the .mtl file in the same diretory as the texture files and the .mtl is targeting a non-existing child folder.  

_May support creating own child folders in the future._

# How to use
- Selet your current .mtl file
- Select where to save the new .mtl file
